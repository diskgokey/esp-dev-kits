ESP-Module-Prog-SUB-02
======================

:link_to_translation:`en:[English]`

ESP-Module-Prog-SUB-02 是专为模组设计的烧录子板，无需将模组与电源和信号线焊接便可进行烧录。作为子板，ESP-Module-Prog-SUB-02 无法单独使用，需搭配底板 ESP-Module-Prog-1 或 ESP-Module-Prog-1R 共同使用。

.. note::

    作为子板，ESP-Module-Prog-SUB-02 无法单独使用，需搭配底板 ESP-Module-Prog-1 或 ESP-Module-Prog-1R 共同使用。

有关底板的详细信息，请参见 :doc:`../esp-module-prog-1-r/user_guide`。

ESP-Module-Prog-SUB-02 用于适配部分 ESP-WROOM、ESP32-C3-WROOM、ESP8684-WROOM 及 ESP32-H2-WROOM 系列模组，请参见 :ref:`适配模组列表 <fitting-modules-of-prog-02>`。
